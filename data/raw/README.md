# Raw Data Directory

This directory should contain the following raw data files:

- `Reddit_2021.csv` - Cleaned Reddit comments from 2021 (columns: hour, clean_text)
- `btc_5min_2021.csv` - Bitcoin 5-minute price data for 2021 (columns: datetime, open, high, low, close, volume, log_return)
- `tweets_cleaned.csv` - Training data for SVM sentiment model (optional)

**Note:** Raw data files are excluded from the repository due to file size constraints. Users must obtain these files separately to run the analysis scripts.
